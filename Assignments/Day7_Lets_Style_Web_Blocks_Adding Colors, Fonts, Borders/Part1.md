Please solve using the VS Code, keep the styles in the HTML file (internal styling), once done, please upload this file to the Google Drive and Submit the File Link

**Objective**: Use a combination of inline and internal CSS styling techniques to create a fully customized and visually attractive webpage header, complete with text, images, tables, and various CSS properties.

#### **Instructions**:

1. **Header Section**:

   - Create a **header** section containing:
     - A **main title** in an `<h1>` tag, for example, "Welcome to My Webpage."
     - A **subtitle** in an `<h2>` tag, like "Learning Web Development."
     - Add a **navigation bar** with three links (`<a>` tags) to Home, About, and Contact pages. (hint: use`nav` tag)

2. **Styling**:

   - Use **inline CSS** to:
     - Set the background color of the `<h1>` to **light grey** and change the font color to **dark blue**.
     - Make the text of the `<h2>` bold and change its color to **green**.
     - Give the `<a>` links a different hover effect (underline on hover).
     - Adjust the **font size** of the title to **40px** and subtitle to **28px**.
   - Use **internal CSS** (inside a `<style>` tag) to:
     - Set the **font family** of the entire page to **Arial or Times New Roman**.
     - Add a **3px solid black border** around the header and give it **5px padding**.
     - Set the **background color** of the webpage to a light gradient or a solid light color of your choice.
     - Center-align all text in the header using CSS.

3. **Image**:

   - Add an image below the header section.
   - Use an image of your choice and:
     - Set its width to **400px**.
     - Apply `border-radius: 50%` to make the image **circular**.
     - Add a **shadow** around the image using the `box-shadow` property.

4. **Table**:

   - Below the image, create a **table** that contains at least 3 rows and 2 columns (e.g., a list of services or sample data).
   - Style the table using **CSS**:
     - Add **border** to the table and set the **border-collapse** property.
     - Add different **border colors** to each table cell.
     - Use **alternating row background colors** for better readability (`nth-child()` selector).

5. **Buttons**:

   - Below the table, add a **button** labeled “Learn More” and style it:
     - Use CSS to apply a **background color**, **text color**, and **padding**.
     - Add a **hover effect** (e.g., change the background color on hover).

6. **Paragraph Section**:

   - Below the button, add a paragraph describing the website or services offered.
   - Style the paragraph with:
     - **Text alignment** set to justify.
     - **Text color** different from the default (choose a color like dark grey).
     - Add a **2px solid border** around the paragraph.

7. **Form**:

   - Add a **simple form** at the bottom to collect the visitor’s name and email:
     - Use the `<input>` tag to create fields for **name** and **email**.
     - Use a **button** to submit the form (no need to make it functional).

8. **Footer Section**:

   - At the bottom of the page, add a **footer** containing a copyright statement.
   - Style the footer with:
     - A **background color** different from the rest of the page (like dark grey).
     - **Font size** smaller than the main text.
     - Center-align the footer text.
